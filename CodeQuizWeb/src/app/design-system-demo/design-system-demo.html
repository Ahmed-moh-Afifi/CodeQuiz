CodeQuiz Design System

A comprehensive design system ported from the MAUI desktop app. Each section shows both CSS class and Angular component usage via tabs. Click </> to see code snippets.

Quick Start Guide

1. Import the design system

// In your styles.scss
@use 'styles/design-system';

2. Import components

import { CqButton, CqCard,
  CqInput, CqDialog
} from './design-system';

3. Theme support

document.documentElement
  .setAttribute('data-theme', 'light');
// Default is dark theme

4. Use components

<cq-button type="primary">
  Click Me
</cq-button>
<cq-card type="elevated">
  <span slot="card-title">Hi</span>
  Content here
</cq-card>
